# AI Project Kickoff Template
**Version:** 1.0  
**Author:** Nigel Hickey  
**Website:** https://nigelhickey.com

---

## Purpose

Describe what this project is for in one or two sentences.

Example:

> This project helps me plan family travel without digging through old messages, reservations, notes, and websites every time we go somewhere.

---

# What Success Looks Like

What does success look like for this project?

Examples:

- Complete a family vacation plan
- Publish a series of blog posts
- Finish researching a major purchase
- Organize work documentation

---

# Current Goal

What are you trying to accomplish right now?

Keep this updated as the project evolves.

---

# What the AI Should Know

List any background information that will help the AI understand your situation.

Examples:

- Personal preferences
- Existing plans
- Important relationships
- Budget
- Constraints
- Previous decisions
- Relevant history

Only include information that improves the quality of the conversation.

---

# Current Priorities

List the things that matter most right now.

Example:

1. Book flights
2. Reserve a hotel
3. Build a daily itinerary
4. Stay within budget

---

# Decisions Already Made

Capture decisions so they don't have to be made again.

Example:

- Flying from Dallas
- Staying in the city center
- Budget capped at $2,500
- Three-night trip
- Hotel preferred over Airbnb

---

# Project Rules

Tell the AI how you want it to behave.

Example:

- Treat approved decisions as intentional.
- Don't reopen decisions unless I ask.
- Point out contradictions when they appear.
- Suggest improvements only when appropriate.
- Build on previous work instead of starting over.

---

# What Belongs In This Project

Examples:

- Research
- Notes
- Drafts
- Plans
- Decisions
- Links
- Checklists
- Brainstorming related to this project

---

# What Stays Out

Examples:

- Unrelated projects
- Random brainstorming
- Temporary conversations
- Personal information that doesn't improve the outcome

Keeping projects focused makes them easier to manage over time.

---

# Parking Lot

Capture ideas that are worth remembering but don't belong in the current task.

Example:

- Research another destination
- Compare airline rewards later
- Create a packing checklist
- Write a follow-up blog post

Don't interrupt today's work to chase tomorrow's idea.

---

# Privacy Reminder

Treat AI like a trusted coworker, not a vault.

Good things to share:

- Goals
- Preferences
- Plans
- Project history
- Research
- Context

Avoid sharing:

- Passwords
- Banking credentials
- Recovery codes
- Government ID numbers
- Sensitive information that belongs to someone else

Use the minimum effective context needed to accomplish the task.

---

# First Prompt

Once you've completed this template, your first prompt can be as simple as:

> Read this project and use it as the foundation for all future conversations. Build on previous decisions instead of starting over. If something is unclear, ask questions before making assumptions.

---

# Notes

This template is intentionally simple.

Don't try to build the perfect AI system on day one.

Start with one real project.

Capture enough context to stop repeating yourself.

Improve it over time.
