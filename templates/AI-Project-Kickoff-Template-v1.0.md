## How to Use This Template

Copy this file into a new project in ChatGPT, Claude, or whatever assistant you use. Work through it top to bottom and fill in each section with your own details.

Here is the one convention to know. Every fill-in section starts with a short prompt in *italics*, and some include an **Example** block. Those two things are guidance, not your content. Replace them with your own answers and delete the parts you don't use. When you're done, the file should hold only your project's real details.

You don't need every section, and you don't need to fill it out perfectly. Enough context to stop repeating yourself is the whole goal. Keep whatever helps you, including the Privacy Reminder and First Prompt near the end, and delete this How to Use section once your template is set up.

> **Example:**
> - Treat approved decisions as intentional.
> - Don't reopen decisions unless I ask.
> - Point out contradictions when they appear.
> - Suggest improvements only when appropriate.
> - Build on previous work instead of starting over.

## What Stays Out

*What should stay out, so the project stays focused?*

> **Examples:**
> - Unrelated projects
> - Random brainstorming
> - Temporary conversations
> - Personal information that doesn't improve the outcome

Keeping projects focused makes them easier to manage over time.

---

## Parking Lot

*Capture ideas that are worth remembering but don't belong in the current task.*

> **Example:**
> - Research another destination
> - Compare airline rewards later
> - Create a packing checklist
> - Write a later blog post

Don't interrupt today's work to chase tomorrow's idea.

---

## Privacy Reminder

Treat AI like a trusted coworker, not a vault.

Good things to share:

- Goals
- Preferences
- Plans
- Project history
- Research
- Context

Avoid sharing:

- Passwords
- Banking credentials
- Recovery codes
- Government ID numbers
- Sensitive information that belongs to someone else

Use the minimum effective context needed to accomplish the task.

---

## First Prompt

*Once you've filled in this template, your first prompt can be as simple as this.*

> Read this project and use it as the foundation for all future conversations. Build on previous decisions instead of starting over. If something is unclear, ask questions before making assumptions.

---

## Notes

This template is intentionally simple.

Don't try to build the perfect AI system on day one. Start with one real project. Capture enough context to stop repeating yourself. Improve it over time.
